## `aido-demo.gif` (to record)

A short (~15–25s) looping demo shown in the README hero. Suggested shot list:

1. An AI-authored PR opens (or type `aido review` in a PR comment box).
2. The GitHub Actions run kicks off (keep this brief).
3. Aido posts its review — scroll the inline suggestions + faceted notes.

Keep it tight and loop the payoff (the posted comment), not the waiting. Target
width ~820px to match the README `<img>`. Tools: [Kap](https://getkap.co/),
[LICEcap](https://www.cockos.com/licecap/), or `ffmpeg` from a screen capture.
